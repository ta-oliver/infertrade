# InferTrade

InferTrade is an Apache 2.0-licensed Python package for researching trading signals,
strategy allocations, and portfolio performance analytics.

## What InferTrade provides

- Trading signal and allocation algorithms
- Utilities for portfolio performance and export workflows
- API helpers for InferTrade API integrations
- Example notebooks and test coverage for core behavior

## Quick start

```bash
python3 -m pip install -r requirements.txt
python3 -m pip install -r requirements-dev.txt
make test
```
